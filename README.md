# 🚀 Data Structure Visualizer

An interactive **web based Data Structure Visualizer** that helps users understand complex data structures and algorithms through **real time animations and visual feedback**.

Built using **HTML, CSS, and JavaScript** with powerful animation libraries like **Anime.js** and **Three.js**.

---

## 📌 Features

### 🌳 Binary Heap
- Min Heap & Max Heap support  
- Insert & Extract operations  
- Step by step animation (bubble up / bubble down)  
- Array representation visualization  
- Detailed operation logs  

---

### 🌲 AVL Tree (Self Balancing BST)
- Insert & Delete operations  
- Automatic balancing with rotations  
- Height & balance factor visualization  
- Real time animation of rotations  

---

### 🔗 Graph Algorithms
- Create directed / undirected graphs  
- Add / remove vertices and edges  
- Weighted edges supported  

#### Algorithms included:
- Breadth First Search (BFS)  
- Depth First Search (DFS)  
- Dijkstra’s Algorithm  
- Prim’s Minimum Spanning Tree  

---

### #️⃣ Hash Table
- Insert & Search operations  
- Collision handling using **Linear Probing**  
- Hash function: `key % size`  

---

## 🎮 Controls / Navigation

### 🖥️ Desktop:
- ⬅️ **Left Arrow** → Navigate left  
- ➡️ **Right Arrow** → Navigate right  
- ⏎ **Enter / Click** → Select data structure  

### 📱 Mobile:
- 👈 Swipe left/right to navigate  
- 👆 Tap / Double tap to select  

---

## 🛠️ Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla JS)**  

### Libraries:
- **Anime.js** for animations  
- **Three.js** for 3D rendering  

## 🎮 How to Use

1. Open the project in your browser  
2. On the landing page:
   - Swipe to navigate between data structures  
   - Double tap to select  
3. Use the control panel to:
   - Insert values  
   - Run algorithms  
   - Modify structures  
4. Watch real time animations and logs
